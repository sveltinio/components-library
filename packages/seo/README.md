## Description

A collection of SEO related Svelte components for metatags, json-ld.

## Components List

### Meta

- PageMetatags
- OpenGraph
- TwitterCard

### SchemaOrg

- JsonLdWebSite
- JsonLdWebPage
- JsonLdBreadcrumbs
- JsonLdSiteNavigationElement

## Install

```bash
# pnpm
pnpm install @sveltinio/seo

# npm
npm install @sveltinio/seo

# yarn
yarn add @sveltinio/seo
```

## Demos

```bash
pnpm dev
```
