## Description

A collection of essentials Svelte _unstyled_ components.

## Components List

- [Button]
- [ButtonGroup]
- [ColorViewer]
- [Dropdown]
- [Link]
- [Tabs]
- [ToggleList]
- [ScrollToTopButton]

## Install

```bash
# pnpm
pnpm install @sveltinio/essentials

# npm
npm install @sveltinio/essentials

# yarn
yarn add @sveltinio/essentials
```

## Demos

```bash
pnpm dev
```
