## Description

A collection of _essentials_, type-safe, accessible and configurable Svelte components.

## Components List

- [Button]
- [ButtonGroup]
- [ColorViewer]
- [Dropdown]
- [Link]
- [Tabs]
- [ToggleList]

## Install

```bash
# pnpm
pnpm install @sveltinio/essentials

# npm
npm install @sveltinio/essentials

# yarn
yarn add @sveltinio/essentials
```

## Demos

```bash
pnpm dev
```
